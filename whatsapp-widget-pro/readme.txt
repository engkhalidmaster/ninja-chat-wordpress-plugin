إضافة احترافية لإضافة ويدجت WhatsApp لموقع ووردبريس مع دعم تحليلات جوجل

== Description ==

WhatsApp Widget Pro هي إضافة احترافية تتيح لك إضافة ويدجت WhatsApp تفاعلي لموقع ووردبريس الخاص بك. تأتي الإضافة بمميزات متقدمة مثل:

**المميزات الرئيسية:**

* ويدجت دردشة عائم مع تصميم احترافي
* دعم فرق متعددة مع أرقام WhatsApp مختلفة
* رسائل ترحيب قابلة للتخصيص
* تتبع الأحداث مع Google Analytics
* إحصائيات مفصلة لاستخدام الويدجت
* واجهة إدارة سهلة الاستخدام
* تصميم متجاوب لجميع الأجهزة
* دعم كامل للغة العربية

**تتبع Google Analytics:**

* تتبع فتح نافذة الدردشة
* تتبع بدء المحادثات مع الفريق
* إحصائيات مفصلة حول معدلات التحويل
* تقارير مخصصة لكل قسم

**سهولة الاستخدام:**

* تثبيت بنقرة واحدة
* إعداد سريع في دقائق
* واجهة إدارة باللغة العربية
* دعم فني متواصل

== Installation ==

1. ارفع ملفات الإضافة إلى مجلد `/wp-content/plugins/whatsapp-widget-pro/`
2. فعل الإضافة من خلال قائمة 'إضافات' في ووردبريس
3. اذهب إلى 'الإعدادات' > 'WhatsApp Widget' لإعداد الإضافة
4. أضف أرقام فريقك وخصص الرسائل
5. فعل تتبع Google Analytics إذا رغبت في ذلك

== Frequently Asked Questions ==

= هل تدعم الإضافة اللغة العربية؟ =

نعم، الإضافة مصممة خصيصاً لدعم اللغة العربية بشكل كامل.

= هل يمكنني إضافة أكثر من رقم WhatsApp؟ =

نعم، يمكنك إضافة عدد غير محدود من أرقام الفريق مع تصنيفهم حسب الأقسام.

= هل تدعم الإضافة Google Analytics؟ =

نعم، الإضافة تدعم تتبع الأحداث مع Google Analytics مع إحصائيات مفصلة.

= هل الإضافة متجاوبة مع الأجهزة المحمولة؟ =

نعم، الويدجت مصمم ليكون متجاوب بالكامل مع جميع أحجام الشاشات.
